# Анализ результатов A/B-тестирования

## 1. Описание данных
В исследовании используются данные из файла `Marketing_AB.csv` объемом 588 101 запись. Набор данных содержит следующие признаки:
* `user id` — уникальный идентификатор пользователя.
* `test group` — группа пользователя (`ad` — тестовая группа с показом рекламы, `psa` — контрольная группа с показом социальной рекламы).
* `converted` — статус совершения конверсии (`True` / `False`).
* `total ads` — суммарное количество увиденных пользователем рекламных объявлений.
* `most ads day` — день недели, в который пользователь увидел наибольшее число объявлений.
* `most ads hour` — час суток, на который пришлось пиковое количество показов объявлений.

## 2. Проверка гипотез
Для оценки статистической значимости результатов сформулированы следующие гипотезы:
* **Нулевая гипотеза ($H_0$):** Конверсия в тестовой группе `ad` равна конверсии в контрольной группе `psa` ($p_{ad} = p_{psa}$).
* **Альтернативная гипотеза ($H_1$):** Конверсии в группах статистически значимо различаются ($p_{ad} \neq p_{psa}$).

Уровень значимости установлен на уровне $\alpha = 0.05$. Статистическая проверка долей выполняется с помощью двухвыборочного Z-теста.

## 3. Результаты статистического анализа
* **Размеры выборок:** группа `ad` — 564 577 пользователей, группа `psa` — 23 524 пользователя.
* **Показатели конверсии:**
  * Группа `ad`: **2.5547%** (95% ДИ: 2.5135% – 2.5958%).
  * Группа `psa`: **1.7854%** (95% ДИ: 1.6162% – 1.9546%).
* **Статистические показатели:**
  * Абсолютная разность долей: **0.7692%** (95% CI: 0.5951% – 0.9434%).
  * Относительный прирост (Lift): **+43.1%**.
  * Z-статистика = **7.37**.
  * $p\text{-value}$ = **$1.71 \times 10^{-13}$**.

Поскольку $p\text{-value} < \alpha$, нулевая гипотеза отвергается. Эффект от показа рекламы является статистически значимым.

## 4. Анализ зависимости конверсии от количества показов
Пользователи из группы `ad` были сегментированы по группам частоты показов (`freq_bucket`):
* Интервал 1–5 показов: конверсия составляет **0.25%**.
* Интервал 101+ показов: конверсия составляет **17.14%**.

Данные показывают монотонный рост уровня конверсии по мере увеличения общего числа просмотров рекламы пользователем.
